# Evidencia - Mapeo de campos Tickets API

Fecha: 2026-03-15

## Objetivo
Alinear el mapeo de campos de tickets con la vista `vw_tickets_operacion` del backend.

## Campos validados
- `categoria_ticket` -> UI (categoria)
- `activo_serial` -> UI (dispositivo)
- `tecnico_asignado` -> UI (asignado a)
- `usuario_reporta` -> UI (creado por)
- `fecha_creacion` -> UI (fecha)

## Resultado
Pendiente de validacion con datos reales en entorno de integracion.
